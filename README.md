# ⏰ Digital Clock Web App

<p align="center">
  <b>A real-time digital clock built using HTML, CSS, and JavaScript 🚀</b>
</p>

---

## ✨ Features

⏰ Displays current time in real-time
⚡ Updates every second
📱 Responsive design for all devices
🎨 Clean and minimal UI
🌙 Supports 12-hour / 24-hour format (optional)

---

## 🛠️ Tech Stack

* 🧱 HTML5 – Structure
* 🎨 CSS3 – Styling
* ⚡ JavaScript – Time logic

---

## ⚙️ How It Works

* JavaScript fetches the current system time
* Updates the display every second using `setInterval`
* Shows time in hours, minutes, and seconds
